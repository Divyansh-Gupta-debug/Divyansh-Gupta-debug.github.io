How It Works

Up and running in 30 seconds

1

Install the Extension

Click "Add to Chrome" and install Classroom Pro from the Chrome Web Store. No sign-up required.

2

Open Google Classroom

Navigate to any class. The search bar appears automatically at the top of the page.

3

Start Searching & Pinning

Type anything to search posts instantly. Click the Pin button on any post to save it for quick access later.

4

Use Deep Search for Documents

Toggle Deep Search and press Enter to search inside all attached files — PDFs, Docs, Slides and more.

🔒 Your Privacy Comes First

Classroom Pro stores all data locally in your browser using Chrome's built-in storage. Nothing is ever transmitted to an external server.

The extension only requests read-only access to your Classroom and Drive data — it cannot modify, delete, or create any content.

No analytics. No ads. No data collection. Period.
